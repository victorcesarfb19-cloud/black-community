# GUIA DE DEPLOY - BLACK COMMUNITY

## 🌐 LINKS PARA ACESSAR SEU SITE

### Opção 1: VERCEL (Recomendado) ⭐

**Passo 1:** Acesse [vercel.com](https://vercel.com)

**Passo 2:** Clique em "New Project"

**Passo 3:** Conecte seu repositório GitHub `victorcesarfb19-cloud/black-community`

**Passo 4:** Clique em "Import"

**Resultado:** Seu site estará em:
```
🔗 https://black-community.vercel.app
```

### Opção 2: Com Domínio Customizado

1. Compre um domínio (GoDaddy, Namecheap, etc)
2. Em Vercel: Settings → Domains
3. Adicione seu domínio
4. Seu site estará em:
```
🔗 https://seudominio.com
```

### Opção 3: NETLIFY

1. Acesse [netlify.com](https://netlify.com)
2. "New site from Git" → Selecione seu repositório
3. Deploy automático

```
🔗 https://black-community-seu-nome.netlify.app
```

## ✅ Checklist Pré-Deploy

- [ ] Código commitado: `git push origin landing-page-v1`
- [ ] Sem erros no build: `npm run build`
- [ ] Testes locais passando: `npm run dev`
- [ ] Versionamento correto no package.json
