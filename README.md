# BLACK COMMUNITY - Premium Landing Page

🚀 **Plataforma Digital Exclusiva para Crescimento Acelerado**

## 📋 Sobre o Projeto

Landing page premium de alta conversão para a BLACK COMMUNITY - uma comunidade privada com acesso a cursos, ferramentas IA, templates e recursos exclusivos.

### ✨ Características Principais

- ✅ Design Premium com Glassmorphism
- ✅ Animações Suaves com Framer Motion
- ✅ Responsivo (Mobile-First)
- ✅ Next.js 14 + TypeScript
- ✅ Tailwind CSS Personalizado
- ✅ SEO Otimizado
- ✅ Performance Máxima (Core Web Vitals)
- ✅ Dark Mode Premium

## 🎯 Seções da Landing Page

1. **Hero** - Headline impactante com CTA principal
2. **Features** - 6 recursos principais em cards interativos
3. **Biblioteca** - Categorias de conteúdo (45+ cursos, 120+ templates, etc)
4. **Cursos** - 6 cursos premium com detalhes
5. **Ferramentas** - Softwares e plataformas inclusos
6. **Timeline** - 4 passos do processo de onboarding
7. **Benefícios** - 6 vantagens principais
8. **Depoimentos** - 3 testimoniais de membros
9. **FAQ** - 5 perguntas frequentes com accordion
10. **CTA Final** - Call-to-action para conversão
11. **WhatsApp Button** - Botão flutuante para contato

## 📦 Instalação & Setup

### 1. Clone o Repositório
```bash
git clone https://github.com/victorcesarfb19-cloud/black-community.git
cd black-community
```

### 2. Instale as Dependências
```bash
npm install
```

### 3. Rode o Servidor de Desenvolvimento
```bash
npm run dev
```

Acesse em: **http://localhost:3000**

### 4. Build para Produção
```bash
npm run build
npm run start
```

## 🚀 Deploy (Vercel)

### Opção 1: Via GitHub Integration (MAIS FÁCIL)
1. Acesse [vercel.com](https://vercel.com)
2. Conecte seu repositório GitHub
3. Clique em "Import"
4. Vercel fará deploy automático

### Opção 2: Via Vercel CLI
```bash
npm install -g vercel
vercel
```
